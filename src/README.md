## 🚀 実行方法

以下のように `MemoApp` を起点に実行してください：

```bash
cd src
javac app/MemoApp.java cli/MemoCli.java memo/MemoManager.java
java app.MemoApp
```

または、VSCode上で `MemoApp.java` を開き、右上の「▶」ボタンから直接実行可能です。
